\# ⏰ Digital Clock Dashboard



A dynamic \*\*JavaScript Digital Clock\*\* that displays real-time information including time, greeting, date, timezone, progress bars for day/month/year, and a daily motivational quote.



\## Features



\* Real-time digital clock

\* Greeting based on time of day

\* Current date display

\* Timezone detection

\* Day progress percentage

\* Progress bars for:



&nbsp; \* Day completion

&nbsp; \* Month completion

&nbsp; \* Year completion

\* Week number display

\* Quote of the day

\* Dynamic background (morning / afternoon / evening / night)

\* Dark / Light mode toggle



\## Technologies Used



\* HTML

\* CSS

\* JavaScript
